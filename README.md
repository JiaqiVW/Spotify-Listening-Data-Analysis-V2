# Spotify-Listening-Data-Analysis
## Version 1 - V1
Collected and processed Spotify track and user streaming data in Python, conducted exploratory analysis on listening patterns, applied NLP (bag-of-words) to song titles, tested additional classification and regression models, further evaluated using Principal Component Analysis, and implemented a collaborative filtering recommendation system.

## Version 2 - V2
Collected and processed Spotify track and user streaming data in Python, conducted exploratory analysis on listening patterns, applied methods like KNN, Logistic Regression, SVM, and Random Forest to predict whether songs will be high-streaming or low-streaming.
